http-cat web component

This is how you embed an HTTP Cat using web components.

<http-cat status="404"></http-cat>

And it looks like this:

API

Available attributes

  • status - The HTTP status code to show the related cat image

This component is just an img tag so it supports all standard img element attributes (don't forget alt!).
